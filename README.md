# Galaxy Maze Guardian game

## A maze game app developed with React.

### Getting started

Do the following:

clone this repo to your local directory.
`git clone https://github.com/FitoGT/maze-guardian.git`

navigate to directory
`cd maze-guardian`

#### run

`npm install`
to install the dependencies

#### run test

`npm test`

#### run

`npm start`

to start the server
